# Sarangani Study Center

A web application designed to support the daily operations of the Sarangani Study Center. It provides a public-facing view along with an internal system for meal logging and tracking for both residents and administrators.

### To Access

- **Public:** https://sarangani-center.netlify.app/
- **Admin:** https://sarangani-center.netlify.app/admin-dashboard
- **Resident:** https://sarangani-center.netlify.app/resident-dashboard 

## Features

- Public View
    - Accessible interface for general users
    - Displays relevant information about the study center
- Meal Logging System
    - Residents can log their daily meals
    - Track attendance and meal participation
- Admin Dashboard
    - Monitor meal data across residents
    - Manage and review meal logs efficiently
    - Alert residents on their meal logs


## Tech Stack

**Client:** React, TailwindCSS

**Deployment:** Netlify

**Server:** Firebase (BaaS)

**Pair Agent:** Claude Code, Github Copilot
